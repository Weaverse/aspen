_Aspen is a Shopify Hydrogen theme built with Weaverse, the theme is suitable for furniture and home decor stores._

## Demo

- Live store: https://weaverse-aspen-demo.fly.dev
- Customizing Aspen on Weaverse Studio: https://studio.weaverse.io/demo?theme=aspen

## What's included

- Remix
- Hydrogen
- Oxygen
- Shopify CLI
- Biome (Eslint, Prettier alternative)
- GraphQL generator
- TypeScript and JavaScript flavors
- Tailwind CSS (via PostCSS)
- Radix UI
- New Shopify customer account API
- Full-featured setup of components and routes
- Fully customizable inside [Weaverse Studio](https://weaverse.io)
